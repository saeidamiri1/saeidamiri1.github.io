# Emphasis 
*Italic text*
How:
```
 *Italic text* or _Italic text_
```
**Bold text**
How:
```
 **Bold text** or __Bold text__
```

~~Strikethrough~~
How:
```
~~Strikethrough~~
```

> blockquote
>> blockquote
How:
```
> blockquote
>> blockquote
```

<span style="color:red">Color</span>

How: 
```
<span style="color:red">Color</span>
```

[Link](http://www.google.com/)

```
[Link](http://www.google.com/)
```

# List

Unordered List
- Bullet
  - sub bullet
    - Sub-sub bullet
- Bullet
  - sub bullet
    - Sub-sub bullet

How:
```
Unordered List
- Bullet
  - sub bullet
    - Sub-sub bullet
- Bullet
  - sub bullet
    - Sub-sub bullet
```

1. Numbered list
    1. Sub numbered list
    2. Sub numbered kist


How:
```
1. Numbered list
    1. Sub numbered list
    2. Sub numbered kist
```

Task List
- [ ] An uncompleted task
- [x] A completed task

How:
```
- [ ] An uncompleted task
- [x] A completed task
```

_Horizontal line :_

-----

How:
```
-----
```
